# Borderline

A fun quiz game where you have to name the country based on its shape!
Think you know China from Chile? Iraq from Iran? 

* Written in Swift
* Uses standard frameworks including:
  * CoreData
  * UIKit
  * StoreKit
  * AVFoundation
  * Auto Layout
* Icon and art design
